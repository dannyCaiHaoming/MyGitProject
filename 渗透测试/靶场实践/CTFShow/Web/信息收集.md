#### .git源码泄露

在创建`git`仓库的时候，目录下会创建一个`.git`隐藏文件。文件里面会保存仓库更改历史，因此如果项目发布的时候，没有移除`.git`文件，在获取到这个文件之后，是可以恢复源代码的。



##### 1. 验证

```php
# 访问这个路径，如果存在
/.git/config	
```



##### 2. 利用

```
# 查看git历史记录
git log
# 恢复版本
git diff xxxxx
```
